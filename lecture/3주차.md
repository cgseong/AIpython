
## 1. 개요

### 1.1 학습 목표

이 교재는 생성형 AI(ChatGPT 등)를 활용하여 파이썬 프로그래밍을 효율적으로 학습하는 방법을 익히는 것을 목표로 한다.

학습자는 다음 능력을 기르게 된다.

- 생성형 AI를 이용한 **코드 생성**    
- 생성형 AI를 이용한 **코드 리뷰**    
- 생성형 AI를 이용한 **코드 디버깅**    
- AI와 협력하는 **프로그래밍 문제 해결 능력**    

---

# Part 1 생성형 AI와 코딩

## 1장 생성형 AI를 활용한 프로그래밍

### 1.1 생성형 AI란

생성형 AI는 사용자의 질문(프롬프트)을 기반으로 **텍스트, 코드, 이미지 등을 생성하는 인공지능**이다.
코딩 학습에서 생성형 AI는 다음과 같은 역할을 수행할 수 있다.

|활용 영역|설명|
|---|---|
|코드 생성|요구사항을 설명하면 코드를 작성|
|코드 설명|코드 동작을 설명|
|코드 리뷰|코드 개선점 제시|
|디버깅|오류 원인 분석|
|학습 코치|개념 설명 및 예제 제공|

---

### 1.2 AI와 함께 코딩하는 방식

전통적인 코딩 학습

```
문제 → 직접 구현 → 오류 → 검색 → 수정
```

AI 기반 코딩 학습

```
문제 → AI에게 설계 질문 → 코드 생성 → 검토 → 수정
```

# Part 2 코드 생성 실습

## 2장 AI를 이용한 코드 생성

### 2.1 코드 생성의 기본 원리

AI에게 코드를 요청할 때는 **명확한 요구사항을 작성하는 것이 중요하다.**

좋은 프롬프트 예

```
파이썬으로 다음 프로그램을 작성해줘.

기능:
- 사용자에게 숫자를 입력받는다
- 짝수인지 홀수인지 판단한다
- 결과를 출력한다
```

---

### 2.2 실습 1 : 숫자 판별 프로그램 만들기

#### 문제

사용자로부터 숫자를 입력받아 **짝수 또는 홀수를 출력하는 프로그램**을 작성하시오.

---

### Step 1 AI에게 코드 생성 요청

프롬프트 예

```
파이썬으로 사용자에게 숫자를 입력받아 짝수인지 홀수인지 출력하는 코드를 작성해줘.
초보자가 이해하기 쉽게 작성해줘.
```

---

### Step 2 생성된 코드

```python
number = int(input("숫자를 입력하세요: "))

if number % 2 == 0:
    print("짝수입니다.")
else:
    print("홀수입니다.")
```

---

### Step 3 코드 이해하기

|코드|설명|
|---|---|
|input()|사용자 입력|
|int()|문자열을 정수로 변환|
|%|나머지 연산|
|if|조건문|

---

### 실습 문제

다음 기능을 하는 프로그램을 AI와 함께 만들어 보시오.

1. 두 숫자를 입력받기    
2. 큰 수를 출력하기

# Part 3 코드 리뷰

## 3장 AI를 이용한 코드 리뷰

코드 리뷰란 작성된 코드의 **문제점, 개선점, 가독성** 등을 분석하는 과정이다.

AI는 다음과 같은 리뷰를 수행할 수 있다.

- 코드 스타일    
- 성능 개선    
- 버그 가능성    
- 가독성 개선    

---

### 3.1 실습 코드

```python
a=input("숫자:")
b=input("숫자:")

if a>b:
 print(a)
else:
 print(b)
```

---

### 3.2 AI에게 코드 리뷰 요청

프롬프트

```
다음 파이썬 코드를 리뷰해줘.
초보자가 이해할 수 있게 개선점도 설명해줘.
```

---

### 3.3 AI 리뷰 결과 예

문제점
1. 입력값이 문자열이다    
2. 변수명이 의미가 부족하다    
3. 들여쓰기가 불규칙하다    

---

### 개선 코드

```python
num1 = int(input("첫 번째 숫자를 입력하세요: "))
num2 = int(input("두 번째 숫자를 입력하세요: "))

if num1 > num2:
    print("큰 수:", num1)
else:
    print("큰 수:", num2)
```

# Part 4 코드 디버깅

## 4장 AI를 이용한 디버깅

디버깅은 프로그램의 **오류를 찾아 수정하는 과정**이다.

AI는 다음을 도와줄 수 있다.
- 오류 원인 분석    
- 수정 코드 제안    
- 디버깅 방법 설명    

---

### 4.1 오류 코드

```python
numbers = [1,2,3,4,5]

for i in range(6):
    print(numbers[i])
```

---

### 실행 오류

```
IndexError: list index out of range
```

---

### AI에게 디버깅 요청

프롬프트

```
다음 파이썬 코드에서 발생하는 오류를 설명하고 수정해줘.
```

---

### AI 분석

문제 원인

```
리스트 길이는 5인데 range(6)으로 접근하고 있음
```

---

### 수정 코드

```python
numbers = [1,2,3,4,5]

for i in range(len(numbers)):
    print(numbers[i])
```

또는

```python
numbers = [1,2,3,4,5]

for num in numbers:
    print(num)
```

# Part 5 AI 기반 코딩 프로젝트

## 5장 미니 프로젝트

### 프로젝트: 성적 평균 계산기

기능

1. 학생 이름 입력    
2. 3개의 점수 입력    
3. 평균 계산    
4. 결과 출력    

---

### 예시 코드

```python
name = input("학생 이름: ")

score1 = int(input("점수1: "))
score2 = int(input("점수2: "))
score3 = int(input("점수3: "))

avg = (score1 + score2 + score3) / 3

print(name, "학생의 평균 점수:", avg)
```
